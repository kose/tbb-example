
# Exsample of Intl TBB:Threading Building Blocks / OpenCV.

## これは？ 
Intel TBB を使った並列化のサンプルです。

- 空間方向：フレーム内のソーベルフィルタ
- 時間方向：フレーム処理をパイプラインで並列化
